## Практичне завдання
1. Розділитися на групи по 4 особи
2. Створити один спільний репозиторій на GitHub
3. Додати файл readme, в який внести список ПІБ та груп членів команди
4. Усім потрібно клонувати цей репозиторій
5. Кожний член команди повинен створити один окремий html-файл і наповнити його контентом. При цьому використовуйте окремі гілки.
6. Запушити все у репозиторій на GitHub.
7. Створити файл index.html, де розмістити посилання на створені окремі html-файли.
8. Запушити все у репозиторій на GitHub.
9. Створити штучно конфлікт, змінивши один і той самий рядок усіма членами команди.
10. Вирішити конфлікти і запушити.
11. Відкрити доступ до репозиторія на пошту morozov@ztu.edu.ua
